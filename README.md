🚀 Git Branch Workflow (Master ➝ Main via Pull Request)

This project demonstrates a Git workflow where:

First, I created the master branch

Pushed changes to GitHub

Created a Pull Request from master to main

Successfully merged it

📌 Step 1: Create Master Branch
git checkout -b master

📸 Screenshot:

![Create Master Branch](screenshots/create-master-branch.png)
📌 Step 2: Push Master Branch
git push -u origin master

📸 Screenshot:

![Push Master Branch](screenshots/push-master.png)
📌 Step 3: Create Pull Request (master ➝ main)

Go to GitHub
Click Compare & Pull Request

📸 Screenshot:

![Create Pull Request](screenshots/pull-request.png)
📌 Step 4: Merge Pull Request

Click Merge Pull Request
